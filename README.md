## :gb: English

### Overview

Interactive web app with complete Moodle guide: local development, plugin creation, course standards (GIFT, SCORM, MBZ) and Aruba hosting migration.

### Guide Contents (17 Sections)

**Development & Configuration**
1. Development Environment - Local Moodle setup (Windows/XAMPP)
2. Moodle Structure - Folders and architecture
3. Creating Plugins - Plugin types and base structure
4. Course Standards - GIFT, XML, SCORM, MBZ formats + scorm-course-generator analysis

**Hosting & Migration (Aruba)**
5. Choose Hosting - Requirements and recommended plans
6. Database Setup - MySQL on Aruba
7. Upload Files - FTP/File Manager
8. Installation - Moodle wizard step-by-step
9. Post Configuration - Initial settings
10. SSL Certificate - Free HTTPS
11. First Course - Create and configure
12. User Management - Roles and enrollments
13. Backup - Backup strategies
14. Maintenance - Updates and cron
15. Troubleshooting - Common issues
16. Resources - Useful links
17. Checklist - Completion verification

### Features

- **Dark Mode** - Light/dark theme toggle (`Ctrl/Cmd + D`)
- **Advanced Search** - Full-text search (`Ctrl/Cmd + K`)
- **Interactive Checklist** - Track progress with auto-save
- **Copy Code** - One-click copy on all code blocks
- **Responsive Design** - Optimized for all devices
- **Moodle Theme** - Official orange color scheme (#f98012)

### Quick Start

```bash
git clone https://github.com/fracabu/Guida_Moodle_Aruba.git
cd Guida_Moodle_Aruba
# Open index.html in browser
```

### Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + K` | Open search |
| `Ctrl/Cmd + D` | Toggle dark mode |
| `Esc` | Close search |
