<a name="english"></a>
## :gb: English

### Overview

An interactive web app offering a complete guide for Moodle installation on Aruba hosting. Step-by-step instructions for setting up your e-learning platform.

### Features

- **Step-by-Step Guide** - Detailed installation instructions
- **Aruba Specific** - Tailored for Aruba hosting
- **Interactive Format** - Easy navigation
- **Troubleshooting** - Common issues and solutions
- **Configuration Tips** - Optimize your Moodle setup

### Quick Start

```bash
git clone https://github.com/fracabu/Guida_Moodle_Aruba.git
cd Guida_Moodle_Aruba
# Open index.html in browser
```
